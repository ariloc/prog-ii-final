LeftBigramProbability compute_left_bigram_probability ( dict[Bigram, int]  bigrams)

Computes the probability for each of the word left bigrams in a list of sentences.

This probability \(P(W_i^j|W_{i-1})\) is calculated as the percentage each bigram \((W_{i-1} W_i^j)\) ( \(j = 1,2,\dots,n\)) represents from all bigrams starting in a certain word \(W_{i-1}\):

\[ P(W_i^j|W_{i-1}) = \frac{C(W_{i-1} W_i^j)}{\sum_{k=1}^n C(W_{i-1} W_i^k)} \]

where \(n\) is the number of distinct words in the list of sentences, and \(C(x)\) is the number of occurrences of the n-gram \(x\) in the corpus.

Parameters
bigramsA dictionary with the count of each of the word left bigrams occurring in a list of sentences.
Returns
The probability calculated for each of the left bigrams.

RightBigramProbability compute_right_bigram_probability ( dict[Bigram, int]  bigrams)

Computes the probability for each of the word right bigrams in a list of sentences.

This probability \(P(W_i^j|W_{i+1})\) is calculated as the percentage each bigram \((W_i^j W_{i+1})\) ( \(j = 1,2,\dots,n\)) represents from all bigrams ending in a certain word \(W_{i+1}\):

\[ P(W_i^j|W_{i+1}) = \frac{C(W_i^j W_{i+1})}{\sum_{i=1}^n C(W_i^k W_{i+1})} \]

where \(n\) is the number of distinct words in the list of sentences, and \(C(x)\) is the number of occurrences of the n-gram \(x\) in the corpus.

Parameters

Returns
The probability calculated for each of the right bigrams.

TrigramProbability compute_trigram_probability ( dict[Trigram, int]  trigrams)

Computes the probability for each of the word trigrams in a list of sentences.

This probability \(P(W_i^j|W_{i-1},W_{i+1})\) is calculated as the percentage each trigram \((W_{i-1} W_i^j W_{i+1})\) ( \(j = 1,2,\dots,n\)) represents from all trigrams starting and ending in a pair of words \(W_{i-1}\) and \(W_{i+1}\) respectively:

\[ P(W_i^j|W_{i-1},W_{i+1}) = \frac{C(W_{i-1} W_i^j W_{i+1})}{\sum_{k=1}^n \#(W_{i-1} W_i^k W_{i+1})} \]

where \(n\) is the number of distinct words in the list of sentences, and \(C(x)\) is the number of occurrences of the n-gram \(x\) in the corpus.

Parameters
trigramsA dictionary with the count of each of the word trigrams occurring in a list of sentences.
Returns
The probability calculated for each of the trigrams.

UnigramProbability compute_unigram_probability ( dict[Unigram, int]  unigrams)

Computes the probability for each of the word unigrams in a list of sentences.

This probability \(P(W^j)\) is calculated as the percentage each one \((W^j)\) represents from all the unigrams in a list of sentences.

\[ P(W^j) = \frac{C(W^j)}{\sum_{k=1}^n C(W^k)} \]

where \(n\) is the number of distinct words in the list of sentences, and \(C(x)\) is the number of occurrences of the n-gram \(x\) in the corpus.

Parameters
unigramsA dictionary with a count of each of the word unigrams occurring in a list of sentences.
Returns
The probability calculated for each of the unigrams.
