◆ call_python_script()

void call_python_script ( char *  personName)

Call the python script named as PYTHON_SCRIPT_NAME with personName as an argument, in order to complete a list of sentences with a missing word written by a person.

The script will use the sanitized output file generated by this program to complete the sentences.

Parameters
personNameThe name of the person whose sentences with missing words will be completed by the python script.
