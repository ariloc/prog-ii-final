dict[Ngram, int] extract_ngrams ( list[str]  sentences,
int  n 
)

Extracts and counts how many times a word n-gram occurs among a list of sentences.

Parameters
sentencesA list of sentences where each of the occurring word n-grams will be counted.
nThe size of the word n-gram. For example, if n=1, it counts the amount of unigrams of words in the sentences. However, if n=2, it counts the amount of bigrams of words, and so on.
Returns
A dictionary containing the count of each n-gram occurring in the sentences.
