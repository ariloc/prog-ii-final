Fills in missing words in a list of sentences written by a person, given a precomputed predictions Dataset generated from sentences written by the person.

Each of the sentences received as an argument must have an underscore ('_'), representing the location of the missing word.

Parameters
sentencesA list of sentences with missing words written by a person to be completed.
datasetA precomputed dataset generated from sentences written by the person.
Returns
The list of completed sentences.
